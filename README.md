# Medical Diagnosis Chatbot

Чат-бот для первичной диагностики заболеваний на основе текстового описания симптомов. Пользователь описывает свои симптомы, бот анализирует текст и выдаёт три наиболее вероятных диагноза. Реализован через ВКонтакте.

## Демонстрация работы

![Демонстрация бота](demo_chat_bot.gif)

## Функциональные возможности

- Приём текстового описания симптомов
- Предобработка текста: очистка и лемматизация через spaCy
- Классификация симптомов обученной моделью LinearSVC
- Выдача трёх наиболее вероятных диагнозов

## Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/luba-l/coursework_chatbot.git
cd coursework_chatbot
```

### Установка зависимостей

```bash
pip install -r requirements.txt
python -m spacy download ru_core_news_sm
```

### Настройка бота ВКонтакте

Создайте сообщество ВКонтакте, получите токен и укажите его в файле `.env`.

### Запуск бота

```bash
python src/vk_bot.py
```

## Используемые технологии

- **Python 3.9+**
- **scikit-learn** — модели машинного обучения
- **spaCy** — лемматизация текста
- **XGBoost** — ансамблевый метод
- **vk_api** — интеграция с ВКонтакте
- **pandas, numpy** — работа с данными
- **matplotlib, seaborn** — визуализация

## Модели машинного обучения

Обучены и сравнены пять моделей: LogisticRegression, LinearSVC, MultinomialNB, MLPClassifier, XGBoost. Для каждой рассчитаны метрики на обучающей и тестовой выборках, проведена кросс-валидация и проверка на реальных пользовательских текстах. Лучший результат показала LinearSVC. Модель и TF-IDF векторизатор сохранены в `models/`.
